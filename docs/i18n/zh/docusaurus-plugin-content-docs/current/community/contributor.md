# 贡献指南

Git Commit Log 尽量使用英文。

Pull Request 尽量保持单一，不同语义的代码贡献应拆分多个 Pull Request。

为了让您的 GitHub ID 显示在 Contributor 列表中，别忘了以下设置：

```shell
git config --global user.name "username"
git config --global user.email "GitHub 账号邮箱"
```
